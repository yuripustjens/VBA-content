# ParagraphFormat Properties (Publisher)

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Alignment](paragraphformat.alignment-property-publisher.md)|Returns or sets a  **PbParagraphAlignmentType** constant that represents the alignment for the specified paragraphs. Read/write.|
| [Application](paragraphformat.application-property-publisher.md)|Used without an object qualifier, this property returns an  **[Application](application-object-publisher.md)** object that represents the current instance of Publisher. Used with an object qualifier, this property returns an  **Application** object that represents the creator of the specified object. When used with an OLE Automation object, it returns the object's application.|
| [AttachedToText](paragraphformat.attachedtotext-property-publisher.md)| **True** if the **Font** or **ParagraphFormat** object is attached to a **TextRange** object. If the object is attached to a **TextRange** object, the document will be updated when properties of the object are changed. If the object is not attached, nothing in the document will be changed until the object is applied to a **TextRange** or **Style** object. Read-only **Boolean**.|
| [CharBasedFirstLineIndent](paragraphformat.charbasedfirstlineindent-property-publisher.md)|Returns or sets the value of the first line indent (in East Asian character width). Read/write  **Long**.|
| [FirstLineIndent](paragraphformat.firstlineindent-property-publisher.md)|Returns or sets a  **Variant** that represents the amount of space (measured in points) to indent the first line in a paragraph. Read/write.|
| [KashidaPercentage](paragraphformat.kashidapercentage-property-publisher.md)|Returns or sets a  **Long** indicating the percentage by which kashidas are to be lengthened for the specified paragraphs. Valid values are from 0 to 100. Read/write.|
| [KeepLinesTogether](paragraphformat.keeplinestogether-property-publisher.md)|Sets or returns an  **MsoTriState** that indicates whether all lines in the specified paragraph will remain in the same text box. Read/write.|
| [KeepWithNext](paragraphformat.keepwithnext-property-publisher.md)|Sets or returns an  **MsoTriState** that indicates whether the following paragraph will remain in the same text box as the specified paragraph. Read/write.|
| [LeftIndent](paragraphformat.leftindent-property-publisher.md)|Returns or sets a  **Variant** that represents the left indent value (in points) for the specified paragraphs. Read/write.|
| [LineSpacing](paragraphformat.linespacing-property-publisher.md)|Returns or sets a  **Variant** that represents the line spacing (in number of lines) for the specified paragraphs. Read/write.|
| [LineSpacingRule](paragraphformat.linespacingrule-property-publisher.md)|Returns or sets a  **PbLineSpacingRule** that represents the line spacing for the specified paragraphs. Read/write.|
| [ListBulletFontName](paragraphformat.listbulletfontname-property-publisher.md)|Sets or retrieves a  **String** representing the list bullet font name from the specified paragraphs. Read/write.|
| [ListBulletFontSize](paragraphformat.listbulletfontsize-property-publisher.md)|Sets or retrieves a  **Single** that represents the list bullet font size from the specified paragraphs. Read/write.|
| [ListBulletText](paragraphformat.listbullettext-property-publisher.md)|Returns a  **String** representing the list bullet text from the specified paragraphs. Read-only.|
| [ListIndent](paragraphformat.listindent-property-publisher.md)|Returns or sets a  **Single** that represents the list indent value (in points) for the specified **ParagraphFormat** object. Read/write.|
| [ListNumberSeparator](paragraphformat.listnumberseparator-property-publisher.md)|Sets or retrieves a  **PbListSeparator** constant that represents the list separator of the specified paragraphs. Read/write.|
| [ListNumberStart](paragraphformat.listnumberstart-property-publisher.md)|Sets or retrieves a  **Long** that represents the starting number of a list. Read/write.|
| [ListType](paragraphformat.listtype-property-publisher.md)|Returns a  **PbListType** constant from the specified **ParagraphFormat** object. Read-only.|
| [LockToBaseLine](paragraphformat.locktobaseline-property-publisher.md)|Returns an  **MsoTristate** that represents whether or not text will be positioned along baseline guides. Read/write.|
| [Parent](paragraphformat.parent-property-publisher.md)|Returns an object that represents the parent object of the specified object. For example, for a  **[TextFrame](textframe-object-publisher.md)** object, returns a **[Shape](shape-object-publisher.md)** object representing the parent shape of the text frame. Read-only.|
| [RightIndent](paragraphformat.rightindent-property-publisher.md)|Returns or sets a  **Variant** that represents the right indent (in points) for the specified paragraphs. Read/write.|
| [SpaceAfter](paragraphformat.spaceafter-property-publisher.md)|Returns or sets a  **Variant** that represents the amount of spacing (in points) after one or more paragraphs. Read/write.|
| [SpaceBefore](paragraphformat.spacebefore-property-publisher.md)|Returns or sets a  **Variant** that represents the amount of spacing (in points) before one or more paragraphs. Read/write.|
| [StartInNextTextBox](paragraphformat.startinnexttextbox-property-publisher.md)|Returns or sets an  **MsoTriState** constant that represents whether to always start the selected paragraph in the next linked text box. Read/write.|
| [Tabs](paragraphformat.tabs-property-publisher.md)|Returns a  **[TabStops](tabstops-object-publisher.md)** object representing the custom and default tabs for a paragraph or group of paragraphs.|
| [TextDirection](paragraphformat.textdirection-property-publisher.md)|Returns or sets a  **PbTextDirection** constant indicating the direction in which text flows in the specified paragraph. Read/write.|
| [TextStyle](paragraphformat.textstyle-property-publisher.md)|Returns or sets a  **Variant** that represents the text style applied to a paragraph. Read/write.|
| [UseCharBasedFirstLineIndent](paragraphformat.usecharbasedfirstlineindent-property-publisher.md)|Returns or sets an  **MsoTriState** constant that specifies whether a paragraph is indented using East Asian character width. Read/write.|
| [WidowControl](paragraphformat.widowcontrol-property-publisher.md)|Sets or returns an  **MsoTriState** that represents whether or not the first or last line of the specified paragraph can appear by itself in a text box. Read/write.|
|Name|Description|
