# ShadowFormat.Type Property (Publisher)

Specifies the shadow type of a shape. Read/write.


## Syntax

 _expression_. **Type**

 _expression_A variable that represents a  **ShadowFormat** object.


## Remarks

The  **Type** property value can be one of the ** [MsoShadowType](http://msdn.microsoft.com/library/msoshadowtype-enumeration-office%28Office.15%29.aspx)** constants declared in the Microsoft Office type library.
