# WebCommandButton Properties (Publisher)

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [ActionURL](webcommandbutton.actionurl-property-publisher.md)|Returns or sets a  **String** that represents the URL of the server-side script to execute in response to a Submit button click. Read/write.|
| [Application](webcommandbutton.application-property-publisher.md)|Used without an object qualifier, this property returns an  **[Application](application-object-publisher.md)** object that represents the current instance of Publisher. Used with an object qualifier, this property returns an  **Application** object that represents the creator of the specified object. When used with an OLE Automation object, it returns the object's application.|
| [ButtonText](webcommandbutton.buttontext-property-publisher.md)|Returns or sets a  **String** that represents the text that appears on the face of a Web command button. Read/write.|
| [ButtonType](webcommandbutton.buttontype-property-publisher.md)|Returns or sets a  **PbCommandButtonType** constant that indicates whether a Web command button will clear or submit form data. Read/write.|
| [DataFileFormat](webcommandbutton.datafileformat-property-publisher.md)|Sets or returns a  **PbSubmitDataFormatType** constant that represents the format to use when saving Web form data to a file. Read/write.|
| [DataFileName](webcommandbutton.datafilename-property-publisher.md)|Returns or sets a  **String** that represents the name of the file in which to save data from a Web form. Read/write.|
| [DataRetrievalMethod](webcommandbutton.dataretrievalmethod-property-publisher.md)|Sets or returns a  **PbSubmitDataRetrievalMethodType** that represents the way data from a Web form is processed. Read/write.|
| [EmailAddress](webcommandbutton.emailaddress-property-publisher.md)|Sets or returns a  **String** representing the e-mail address to use when processing Web form data. Read/write.|
| [EmailSubject](webcommandbutton.emailsubject-property-publisher.md)|Sets or returns a  **String** that represents the subject for e-mail messages generated to process Web form data. Read/write.|
| [HiddenFields](webcommandbutton.hiddenfields-property-publisher.md)|Returns a  **WebHiddenFields** object that represents hidden Web fields attached to a Submit command button.|
| [Parent](webcommandbutton.parent-property-publisher.md)|Returns an object that represents the parent object of the specified object. For example, for a  **[TextFrame](textframe-object-publisher.md)** object, returns a **[Shape](shape-object-publisher.md)** object representing the parent shape of the text frame. Read-only.|
| [PostFormData](webcommandbutton.postformdata-property-publisher.md)|Returns or sets an  **MsoTriState** constant indicating whether the specified Web command button control uses the Microsoft Visual Basic **Get** or **Post** method when submitting form data to a Web server. Read/write.|
|Name|Description|
