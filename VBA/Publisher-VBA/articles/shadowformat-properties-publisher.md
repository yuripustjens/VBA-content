# ShadowFormat Properties (Publisher)

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](shadowformat.application-property-publisher.md)|Used without an object qualifier, this property returns an  **[Application](application-object-publisher.md)** object that represents the current instance of Publisher. Used with an object qualifier, this property returns an  **Application** object that represents the creator of the specified object. When used with an OLE Automation object, it returns the object's application.|
| [Blur](shadowformat.blur-property-publisher.md)|Returns or sets a  **Single** that represents the blur level for a shadow format. Read/write.|
| [ForeColor](shadowformat.forecolor-property-publisher.md)|Returns or sets a  **[ColorFormat](colorformat-object-publisher.md)** object representing the foreground color for the fill, line, or shadow. Read/write.|
| [Obscured](shadowformat.obscured-property-publisher.md)|Returns or sets an  **MsoTriState** value indicating whether the shadow of the specified shape appears filled in and is obscured by the shape. Read/write.|
| [OffsetX](shadowformat.offsetx-property-publisher.md)|Returns or sets a  **Variant** value indicating the vertical offset of the shadow from the specified shape. A positive value offsets the shadow below the shape; a negative value offsets it above the shape. Read/write.|
| [OffsetY](shadowformat.offsety-property-publisher.md)|Returns or sets a  **Variant** value indicating the horizontal offset of the shadow from the specified shape. A positive value offsets the shadow to the right of the shape; a negative value offsets it to the left. Read/write.|
| [Parent](shadowformat.parent-property-publisher.md)|Returns an object that represents the parent object of the specified object. For example, for a  **[TextFrame](textframe-object-publisher.md)** object, returns a **[Shape](shape-object-publisher.md)** object representing the parent shape of the text frame. Read-only.|
| [RotateWithShape](shadowformat.rotatewithshape-property-publisher.md)|Returns or sets an  **MsoTriState** constant that represents whether to rotate the shadow when rotating the shape. Read/write.|
| [Size](shadowformat.size-property-publisher.md)|Returns or sets a  **Single** that represents the width of the shadow. Read/write.|
| [Type](shadowformat.type-property-publisher.md)|Specifies the shadow type of a shape. Read/write.|
| [Visible](shadowformat.visible-property-publisher.md)|Returns or sets an  **MsoTriState** constant indicating whether the specified object or the formatting applied to the specified object is visible. Read/write.|
|Name|Description|
