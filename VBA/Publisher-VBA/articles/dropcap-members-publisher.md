# DropCap Members (Publisher)
Represents a dropped capital letter at the beginning of a paragraph.

## Methods



|**Name**|**Description**|
|:-----|:-----|
| [ApplyCustomDropCap](dropcap.applycustomdropcap-method-publisher.md)|Applies custom formatting to the first letters of paragraphs in a text frame.|
| [Clear](dropcap.clear-method-publisher.md)|Removes the dropped capital letter formatting.|
|Name|Description|

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](dropcap.application-property-publisher.md)|Used without an object qualifier, this property returns an  **[Application](application-object-publisher.md)** object that represents the current instance of Publisher. Used with an object qualifier, this property returns an  **Application** object that represents the creator of the specified object. When used with an OLE Automation object, it returns the object's application.|
| [FontBold](dropcap.fontbold-property-publisher.md)|Sets or returns an  **MsoTriState**constant that represents whether the font for a dropped capital letter or WordArt text effect is bold. Read/write.|
| [FontColor](dropcap.fontcolor-property-publisher.md)|Returns or sets a  **[ColorFormat](colorformat-object-publisher.md)** object that represents the color applied to a specified dropped capital letter.|
| [FontItalic](dropcap.fontitalic-property-publisher.md)|Sets or returns an  **MsoTriState**constant that represents whether the font for a dropped capital letter or WordArt text effect is italic. Read/write.|
| [FontName](dropcap.fontname-property-publisher.md)|Sets or returns a  **String** that represents the name of the font applied to a dropped capital letter or WordArt text effect. Read/write.|
| [LinesUp](dropcap.linesup-property-publisher.md)|Returns or sets a  **Long** that represents the number of lines an initial dropped capital letter is raised above the line of text on which it exists. Read/write.|
| [Parent](dropcap.parent-property-publisher.md)|Returns an object that represents the parent object of the specified object. For example, for a  **[TextFrame](textframe-object-publisher.md)** object, returns a **[Shape](shape-object-publisher.md)** object representing the parent shape of the text frame. Read-only.|
| [Size](dropcap.size-property-publisher.md)|Returns or sets a  **Long** that represents the number of lines high to format a dropped capital letter. Read/write.|
| [Span](dropcap.span-property-publisher.md)|Returns or sets a  **Long** that represents the number of letters included in the specified dropped capital letter. Read/write.|
