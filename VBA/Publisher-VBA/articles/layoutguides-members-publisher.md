# LayoutGuides Members (Publisher)
Represents the measurement grid that appears superimposed on publication pages as an aid to laying out design elements.

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](layoutguides.application-property-publisher.md)|Used without an object qualifier, this property returns an  **[Application](application-object-publisher.md)** object that represents the current instance of Publisher. Used with an object qualifier, this property returns an  **Application** object that represents the creator of the specified object. When used with an OLE Automation object, it returns the object's application.|
| [ColumnGutterWidth](layoutguides.columngutterwidth-property-publisher.md)|Returns or sets the width of the column gutters that are used by the  **LayoutGuides** object to aid in the process of laying out design elements. Read/write **Single**.|
| [Columns](layoutguides.columns-property-publisher.md)|Returns or sets a  **Long** that represents the number of guide columns on a page or the number of columns in a text frame. Read/write.|
| [GutterCenterlines](layoutguides.guttercenterlines-property-publisher.md)|Returns or sets a value that specifies whether to add a center line between the columns and rows of the gutter guides in a master page. Read/write  **Boolean**.|
| [HorizontalBaseLineOffset](layoutguides.horizontalbaselineoffset-property-publisher.md)|Returns a  **Single** that represents the horizontal baseline offset of the specified **LayoutGuides** object. Read/Write.|
| [HorizontalBaseLineSpacing](layoutguides.horizontalbaselinespacing-property-publisher.md)|Returns a  **Single** that represents the horizontal baseline spacing of the specified **LayoutGuides** object. Read/write.|
| [MarginBottom](layoutguides.marginbottom-property-publisher.md)|Returns or sets a  **Variant** that represents the amount of space (in points) between the text and the bottom edge of a cell, text frame, or page. Read/write.|
| [MarginLeft](layoutguides.marginleft-property-publisher.md)|Returns or sets a  **Variant** that represents the amount of space (in points) between the text and the left edge of a cell, text frame, or page. Read/write.|
| [MarginRight](layoutguides.marginright-property-publisher.md)|Returns or sets a  **Variant** that represents the amount of space (in points) between the text and the right edge of a cell, text frame, or page. Read/write.|
| [MarginTop](layoutguides.margintop-property-publisher.md)|Returns or sets a  **Variant** that represents the amount of space (in points) between the text and the top edge of a cell, text frame, or page. Read/write.|
| [MirrorGuides](layoutguides.mirrorguides-property-publisher.md)|Returns or sets a  **Boolean** indicating whether Microsoft Publisher creates mirror guide positions for a book fold publication. **True** if Publisher creates mirror guide positions for separate left and right pages in a book fold publication; **False** if the same margin, row, and column guides are applied to all pages in the publication. Read/write.|
| [Parent](layoutguides.parent-property-publisher.md)|Returns an object that represents the parent object of the specified object. For example, for a  **[TextFrame](textframe-object-publisher.md)** object, returns a **[Shape](shape-object-publisher.md)** object representing the parent shape of the text frame. Read-only.|
| [RowGutterWidth](layoutguides.rowgutterwidth-property-publisher.md)|Returns or sets the width of the row gutters that are used by the  **LayoutGuides** object to aid in the process of laying out design elements. Read/write **Single**.|
| [Rows](layoutguides.rows-property-publisher.md)|Sets or returns a  **Long** that represents the number of rows in a layout guide. Read/write.|
| [VerticalBaseLineOffset](layoutguides.verticalbaselineoffset-property-publisher.md)|Returns a  **Single** that represents the vertical baseline offset of the specified **LayoutGuides** object. Read/write.|
| [VerticalBaseLineSpacing](layoutguides.verticalbaselinespacing-property-publisher.md)|Returns a  **Single** that represents the vertical baseline spacing of the specified **LayoutGuides** object. Read/write.|
|Name|Description|
